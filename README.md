# Nexus Chat v2 🚀

**محادثة فورية** — WebSocket + SQLite. لا Firebase، لا حسابات.

## الميزات

- ✅ **محادثة فورية** عبر WebSocket
- ✅ **حفظ الرسائل** في SQLite (تستمر بعد إعادة التشغيل)
- ✅ **دعم الصور** — لصق (Paste) أو رفع (Upload)
- ✅ **حذف رسائلك** — نقرة واحدة
- ✅ **مؤشر الكتابة** — شو عم يكتب
- ✅ **عرض المتصلين** — كم شخص متصل الآن
- ✅ **عكس اتجاه المحادثة** — Flip Layout
- ✅ **حفظ المسودة** — لا تفقد الرسالة أثناء الكتابة
- ✅ **إعادة اتصال تلقائي** — عند انقطاع النت
- ✅ **100% عربي RTL** — واجهة مظلمة (WhatsApp-style)
- ✅ **آمن من XSS** — تعقيم النصوص بالكامل
- ✅ **لا حسابات خارجية** — كل شيء في الكود

## التشغيل المحلي

```bash
npm install
npm start
```

افتح المتصفح على: **http://localhost:3000**

## النشر على Render (مجاناً)

1. ارفع الكود إلى GitHub
2. في [Render Dashboard](https://dashboard.render.com) → New + Web Service
3. اختر GitHub repo
4. **Build Command:** `npm install`
5. **Start Command:** `node server.js`
6. اضغط Create → خلال دقائق يكون الموقع شغال 🎉

## مقارنة مع v1

| الميزة | v1 | v2 |
|--------|:--:|:--:|
| Backend | Firebase (انتهى) | SQLite + WebSocket |
| Authentication | كلمة سر وهمية | اسم فقط (لا حاجة لكلمة) |
| دوام الرسائل | ❌ تختفي بعد الرفريش | ✅ باقية في SQLite |
| مؤشر الكتابة | ❌ | ✅ |
| عرض المتصلين | ❌ | ✅ |
| حفظ المسودة | ❌ | ✅ |
| أمان XSS | ❌ innerHTML مكشوف | ✅ sanitized + escaped |
| استقلالية | Firebase Console | كل شيء 🫵 في الكود |
